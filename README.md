# React + TypeScript + Vite

### Running The Apps Locally 💪🧑‍🏭

- Clone repo `git clone https://github.com/Zavrsni-rad-ict/solidarna-kuhinja.git`
- Open terminal
- Run `npm i`
- Run `npm run dev`
